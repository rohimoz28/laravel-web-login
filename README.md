## Feature

-   Attendance
-   Authentication
-   Forgot Password
-   Image Upload
-   Rest Api
-   JWT for Authentication

## Tools

-   PHP 8.1
-   Laravel 9.x
-   postgreSQL
-   Architecture: Model - Service - Controller - View
-   Bootstrap 5.2
-   Json Web Token
